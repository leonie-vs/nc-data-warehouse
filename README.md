OLAP: Data Warehousing - Star Schema in Postgres Database

- The bash script run-star-schema.sh runs star-schema.sql and saves the outputs to a file called star-schema-output.txt. In order to make run-all.sh executable, run <chmod +x run-star-schema.sh> once. Then, use the command ./run-star-schema.sh from the root directory to execute the SQL queries.

- The bash script run-queries runs queries.sql and saves the outputs to queries.sql.
